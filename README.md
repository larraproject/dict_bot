\ Telegram Dictionary Bot

Бот для поиска определений английских слов через Free Dictionary API.



\ Возможности

\- Поиск определений, примеров использования и частей речи

\- Асинхронная работа, корректная обработка ошибок сети

\- Режимы: Polling (локально) / Webhook (облако)



\Технологии

\- Python 3.10+

\- `python-telegram-bot` v20 (Async)

\- `httpx` (HTTP-клиент)

\- `dotenv` (Безопасное хранение секретов)

\- GitHub + Render (CI/CD и хостинг)



\Установка и запуск локально

1\. Клонируйте репозиторий: `git clone <url>`

2\. Установите зависимости: `pip install -r requirements.txt`

3\. Создайте `.env` и добавьте `TG\_BOT\_TOKEN=<ваш\_токен>`

4\. Запустите: `python bot.py`
